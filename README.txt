 * 邓老师，我是计算机类7班的吴梦奇
 * 此项目功能为：对数学函数进行简单的符号/数值计算，例如求导、积分、化简、方程间四则运算
 * 该项目由 CMake 构建生成，并且由 git 进行项目管理。其中通过 .gitignore 文件指定不跟踪 ./bin 以及 ./build 文件夹
 * 一下内容为一些规范和方向

1.目前仅限于多函数、单变量(多项式、指数、简单三角函数SCT) 符号/数值 计算，具体计算涉及的单元如下(带有 * 的待定)
    (1) 求导
    (2) 积分
    (3) 化简*
    (4) 函数间四则运算(注意除法)*

2.项目结构
    (1) main.c 操纵整个程序流程
    (2) 字符串处理模块
    (3) 词法分析模块
    (4) 求导模块
    (5) 积分模块
    (6) 化简模块
    (7) 函数间四则运算模块
    (8) 必要零件模块
    (9) 菜单模块

3.规范
    严格控制函数、变量等作用域与存储期，用多少给多少
    禁止全局变量，需要长期保存的变量作用域限定为主函数内，调用时传递

4.操作界面与方式
    操作界面为 cmd/powershell 窗口，操作方式为程序引导输入字符或字符串
    输入作为函数的字符串不应该唯一，可以一种函数多种相似表述

5.具体实现
    (1) 输入 命令关键字 及 作为函数的字符串，处理字符串
        例：diff(x^2 + x - 1, 3) 表示对 x^2 + x - 1 求导三次
    (2) 词法分析模块解析出字符串含义，并构建抽象表达式树，传递根节点给功能函数
    (3) 执行功能函数/化简
    (4) 输出
