 * 此项目功能为：对数学函数进行简单的 符号/数值 计算，如求导、积分、化简、方程间四则运算
 * 该项目由 CMake 构建生成，并由 git 进行项目管理
 *   其中通过 CMakeLists.txt 文件描述项目生成规则，.gitignore 文件指定不跟踪 ./bin 以及 ./build 文件夹
 * 项目满足基本错误检测，包括字符串处理、token词法分析、树分析，并通过 wrongPrint() 函数抛出错误(wrongPrintT()相同)
 * 以下内容为规范与方向

1.仅支持多函数、单变量的 符号/数值 计算(如多项式、指数)，具体计算涉及的单元如下(带有 * 为待定)
    (1) 求导
    (2) 积分*
    (3) 化简
    (4) 函数间四则运算*

2.项目结构
    (1) main.c 操纵整个程序流程
    (2) 字符串处理模块
    (3) 词法分析模块
    (4) 求导模块
    (5) 积分模块
    (6) 化简模块
    (7) 函数间四则运算模块
    (8) 必要零件模块
    (9) 菜单模块

3.规范
    严格控制函数、变量作用域与存储期，用多少给多少
    禁止全局变量，需要长期保存的变量作用域限定为主函数内，调用时传递

4.操作界面与方式
    操作界面为 CMD/PowerShell 窗口，操作方式为程序引导输入字符或字符串，Enter结束输入

5.实现
    (1) 输入 命令关键字() 字符串并处理，例：diff(f, num)
    (2) 处理 命令中作为参数的函数表达式 并拆解为 Token单元 存储进数组 
    (3) 使用 Token单元 构建抽象表达式树
    (4) 执行功能函数/化简
    (5) 输出
    语法规则例：diff(x^2 + x - 1, 3) 表示对 x^2 + x - 1 求导三次
